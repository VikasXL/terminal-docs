# entPlugin

## ensure

```js filename="Terminal" copy
ensure entPlugin name:"pluginName"
ensure entPlugin version:"pluginVersion"
```


## remove

```js filename="Terminal" copy
remove entPlugin name:"pluginName"
```


## update

```js filename="Terminal" copy
update entPlugin name:"pluginName"
update entPlugin version:"version"
update entPlugin singleton:"true | false | yes | no"
update entPlugin description:"null | @null | @empty | paragraph"
update entPlugin filterTags:"null | @empty | text1, text2, ..."
```


## print

```js filename="Terminal" copy
print entPlugin name:"pluginName"
```
