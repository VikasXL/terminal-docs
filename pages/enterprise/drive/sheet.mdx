# driveSheet

## ensure

```js filename="Terminal" copy
ensure driveSheet DriveSheetName type:"master | partitioned"
```


## remove

```js filename="Terminal" copy
remove driveSheet DriveSheetName
```


## update

```js filename="Terminal" copy
update driveSheet DriveSheetName driveType:"master | partitioned"
update driveSheet DriveSheetName roles:"null | @empty | Role1, Role2, ..."
update driveSheet DriveSheetName mediaRoles:"null | @empty | Role1, Role2, ..."
update driveSheet DriveSheetName dataPartitionPeriod:"..."
update driveSheet DriveSheetName dataRetentionDuration:"..."
update driveSheet DriveSheetName dataUpdateDuration:"..."
update driveSheet DriveSheetName dataPriority:"..."
```

**`dataPartitionPeriod`**
 : null | Variable of type: dataPartitionPeriod | dataPartitionPeriod value

**`dataRetentionDuration`**
 : null | Variable of type: duration | duration value

**`dataUpdateDuration`**
 : null | Variable of type: duration | duration value

**`dataPriority`**
 : null | Variable of type: dataPriority | dataPriority value


## print

```js filename="Terminal" copy
print driveSheet DriveSheetName
```
