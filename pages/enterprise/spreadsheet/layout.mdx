# spreadsheetLayout

## ensure

```js filename="Terminal" copy
ensure spreadsheetLayout SpreadsheetLayoutName kind:"card | kanban | list | table | calendar"
```


## remove

```js filename="Terminal" copy
remove spreadsheetLayout SpreadsheetLayoutName
```
