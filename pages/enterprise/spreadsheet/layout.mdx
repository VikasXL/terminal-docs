# spreadsheetLayout

## ensure

```js filename="Terminal" copy
ensure spreadsheetLayout SpreadsheetLayoutName kind:"card | kanban | list | table"
```


## remove

```js filename="Terminal" copy
remove spreadsheetLayout SpreadsheetLayoutName
```


## update

### card

update spreadsheetLayout SpreadsheetLayoutName label:"null | @null | @empty | text"
update spreadsheetLayout SpreadsheetLayoutName description:"null | @null | @empty | paragraph"
update spreadsheetLayout SpreadsheetLayoutName toolTipField:"null | Symbol"
update spreadsheetLayout SpreadsheetLayoutName bkgColorField:"null | Symbol"
update spreadsheetLayout SpreadsheetLayoutName allowToSwitchLayouts:"..."
update spreadsheetLayout SpreadsheetLayoutName groupByField:"null | Symbol"
update spreadsheetLayout SpreadsheetLayoutName kind:"tree | tab"
update spreadsheetLayout SpreadsheetLayoutName showSearchBar:"true | false | yes | no"
update spreadsheetLayout SpreadsheetLayoutName categoryFields:"null | @empty | Field1, Field2, ..."
update spreadsheetLayout SpreadsheetLayoutName advanceFilterFields:"null | @empty | Field1, Field2, ..."
update spreadsheetLayout SpreadsheetLayoutName sortByFields:"null | @empty | Field1, Field2, ..."

**`allowToSwitchLayouts`**
 : null | @empty | SpreadsheetLayout1, SpreadsheetLayout2, ...
