# spreadsheetLayoutListItem

## update

```js filename="Terminal" copy
 update spreadsheetLayoutListItem line:"first | second | third | fourth | fifth"
 update spreadsheetLayoutListItem imageCorner:"..."
 update spreadsheetLayoutListItem images:"..."
 update spreadsheetLayoutListItem segment:"first | middle | caption"
 update spreadsheetLayoutListItem source:"field | var"
 update spreadsheetLayoutListItem text:"..."
 update spreadsheetLayoutListItem color:"..."
 update spreadsheetLayoutListItem font:"..."
 update spreadsheetLayoutListItem showLabels:"true | false | yes | no"
```

**`imageCorner`**
null | imageCorner type variable | imageCorner value

**`images`**
null | @empty | gridOrSectionFields | variables ...

**`text`**
null | field1, field2 | text type variable | variable value

**`color`**
color type field | color type variable | variable value

**`font`**
font type field | font type variable | variable value
