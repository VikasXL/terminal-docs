# spreadsheetLayoutCardItem

## update

```js filename="Terminal" copy
update spreadsheetLayoutCardItem line:"first | second | third | fourth | fifth"
update spreadsheetLayoutCardItem imageCorner:"..."
update spreadsheetLayoutCardItem images:"..."
update spreadsheetLayoutCardItem segment:"first | middle | caption"
update spreadsheetLayoutCardItem source:"field | var"
update spreadsheetLayoutCardItem text:"..."
update spreadsheetLayoutCardItem color:"..."
update spreadsheetLayoutCardItem font:"..."
update spreadsheetLayoutCardItem showLabels:"true | false | yes | no"
```

**`imageCorner`**
 : null | imageCorner type variable | imageCorner value

**`images`**
 : null | @empty | gridOrSectionFields | variables ...

**`text`**
 : null | field1, field2 | text type variable | variable value

**`color`**
 : color type field | color type variable | variable value

**`font`**
 : font type field | font type variable | variable value
