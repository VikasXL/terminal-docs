# group

## ensure

```js filename="Terminal" copy
ensure group GroupName
```


## remove

```js filename="Terminal" copy
remove group GroupName
```


## update

```js filename="Terminal" copy
update group GroupName name:"Symbol"
update group GroupName label:"null | @null | @empty | text"
update group GroupName description:"null | @null | @empty | paragraph"
update group GroupName filterTags:"..."
update group GroupName allowPromptAssistant:"true | false | yes | no"
update group GroupName pinnedActions:"null | @empty | Action1, Action2, ..."
update group GroupName pinnedActionsMobile:"null | @empty | Action1, Action2, ..."
update group GroupName groupPermissions:"..."
update group GroupName chatPermissions:"..."
```

**`filterTags`**
 : null | @empty | FilterTag1, FilterTag2, ...

**`groupPermissions`**
 : null | @empty | $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, Role1, Role2, ...

**`chatPermissions`**
 : null | @empty | $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, Role1, Role2, ...
