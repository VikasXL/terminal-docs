# prompt

## ensure

```js filename="Terminal" copy
ensure prompt PromptName
```


## remove

```js filename="Terminal" copy
remove prompt PromptName
```


## update

```js filename="Terminal" copy
update prompt PromptName filterTags:"..."
update prompt PromptName name:"Symbol"
update prompt PromptName action:"null | Action"
update prompt PromptName promptText:"null | @null | @empty | text"
update prompt PromptName reviewBeforeExecuting:"true | false | yes | no"
update prompt PromptName permissionRoles:"null | @empty | Role1, Role2, ..."
```

**`filterTags`**
 : null | @empty | FilterTag1, FilterTag2, ...


## print

```js filename="Terminal" copy
print prompt PromptName
```
