# groupAction

## ensure

```js filename="Terminal" copy
 ensure groupAction action:"Symbol" roles:"..." menuGroup:"null | @null | @empty | text" deviceSizes:"..."
```

**`roles`**
null | @empty | Role1, Role2, $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, ...

**`deviceSizes`**
null | @empty | mobile, tablet, laptop, desktop


## remove

```js filename="Terminal" copy
 remove groupAction action:"Symbol"
```


## print

```js filename="Terminal" copy
 print groupAction action:"Symbol"
```


## goTo


```js filename="Terminal" copy
 goto groupAction argument:"value"
```
