# gridMenuRow

## ensure

```js filename="Terminal" copy
ensure gridMenuRow action:"Symbol" roles:"..."
ensure gridMenuRow action:"Symbol" menuGroup:"null | @null | @empty | text"
ensure gridMenuRow action:"Symbol" deviceSizes:"..."
ensure gridMenuRow action:"Symbol" showMessageTooltip:"true | false | yes | no"
```

**`roles`**
 : null | @empty | Role1, Role2, $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, ...

**`deviceSizes`**
 : null | @empty | mobile, tablet, laptop, desktop


## remove

```js filename="Terminal" copy
remove gridMenuRow action:"Symbol"
```
