# gridMenu

## ensure

```js filename="Terminal" copy
ensure gridMenu action:"Symbol" roles:"..."
ensure gridMenu action:"Symbol" menuGroup:"null | @null | @empty | text"
ensure gridMenu action:"Symbol" deviceSizes:"..."
ensure gridMenu action:"Symbol" showMessageTooltip:"true | false | yes | no"
```

**`roles`**
 : null | @empty | Role1, Role2, $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, ...

**`deviceSizes`**
 : null | @empty | mobile, tablet, laptop, desktop


## remove

```js filename="Terminal" copy
remove gridMenu action:"Symbol"
```


## print

```js filename="Terminal" copy
print gridMenu action:"Symbol"
```
