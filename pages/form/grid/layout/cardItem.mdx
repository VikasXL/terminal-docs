# gridLayoutCardItem

## update

```js filename="Terminal" copy
 update gridLayoutCardItem line:"first | second | third | fourth | fifth"
 update gridLayoutCardItem imageCorner:"..."
 update gridLayoutCardItem images:"..."
 update gridLayoutCardItem segment:"first | middle | caption"
 update gridLayoutCardItem source:"field | var"
 update gridLayoutCardItem text:"..."
 update gridLayoutCardItem color:"..."
 update gridLayoutCardItem font:"..."
 update gridLayoutCardItem showLabels:"true | false | yes | no"
```

**`imageCorner`**
null | imageCorner type variable | imageCorner value

**`images`**
null | @empty | gridOrSectionFields | variables ...

**`text`**
null | field1, field2 | text type variable | variable value

**`color`**
color type field | color type variable | variable value

**`font`**
font type field | font type variable | variable value
