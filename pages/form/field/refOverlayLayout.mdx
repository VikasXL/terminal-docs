# fieldRefOverlayLayout

## update

```js filename="Terminal" copy
update fieldRefOverlayLayout line:"first | second | third | fourth | fifth"
update fieldRefOverlayLayout imageCorner:"..."
update fieldRefOverlayLayout images:"..."
update fieldRefOverlayLayout segment:"first | middle | caption"
update fieldRefOverlayLayout source:"field | var"
update fieldRefOverlayLayout text:"..."
update fieldRefOverlayLayout color:"..."
update fieldRefOverlayLayout font:"..."
update fieldRefOverlayLayout showLabels:"true | false | yes | no"
```

**`imageCorner`**
 : null | imageCorner type variable | imageCorner value

**`images`**
 : null | @empty | gridOrSectionFields | variables ...

**`text`**
 : null | field1, field2 | text type variable | variable value

**`color`**
 : color type field | color type variable | variable value

**`font`**
 : font type field | font type variable | variable value
