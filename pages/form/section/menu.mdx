# sectionMenu

## ensure

```js filename="Terminal" copy
ensure sectionMenu action:"Action" roles:"..."
ensure sectionMenu action:"Action" notAllowedRoles:"..."
ensure sectionMenu action:"Action" menuGroup:"null | @null | @empty | text"
ensure sectionMenu action:"Action" deviceSizes:"..."
ensure sectionMenu action:"Action" showMessageTooltip:"true | false | yes | no"
ensure sectionMenu action:"Action" mappingVar:"null | Variable of type: mapping"
```

**`roles`**
 : null | @empty | $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, Role1, Role2, ...

**`notAllowedRoles`**
 : null | @empty | $Public, $Manager, $GrandManager, $AllManagers, $Assistants, $AllAssistants, $Self, Role1, Role2, ...

**`deviceSizes`**
 : null | @empty | mobile, tablet, laptop, desktop


## remove

```js filename="Terminal" copy
remove sectionMenu action:"Symbol"
```


## print

```js filename="Terminal" copy
print sectionMenu action:"Symbol"
```
