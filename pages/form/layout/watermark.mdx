# formLayoutWatermark

## update

```js filename="Terminal" copy
 update formLayoutWatermark textPattern:"null | text type variable | text value"
 update formLayoutWatermark textFont:"null | font type variable | font value"
 update formLayoutWatermark textPosition:"..."
 update formLayoutWatermark textOpacity:"..."
```

**`textPosition`**
null | themePlacement type variable | themePlacement value

**`textOpacity`**
null | decimal type variable | decimal value
