# formLayoutWatermark

## update

```js filename="Terminal" copy
update formLayoutWatermark textPattern:"..."
update formLayoutWatermark textFont:"..."
update formLayoutWatermark textPosition:"..."
update formLayoutWatermark textOpacity:"..."
```

**`textPattern`**
 : null | Variable of type: text | text value

**`textFont`**
 : null | Variable of type: font | font value

**`textPosition`**
 : null | Variable of type: themePlacement | themePlacement value

**`textOpacity`**
 : null | Variable of type: decimal | decimal value
