# formLayoutWatermark

## update

```js filename="Terminal" copy
update formLayoutWatermark textPattern:"..."
update formLayoutWatermark textFont:"..."
update formLayoutWatermark textPosition:"..."
update formLayoutWatermark textOpacity:"..."
```

**`textPattern`**
 : null | Variable of type: text | Variable value of type: text

**`textFont`**
 : null | Variable of type: font | Variable value of type: font

**`textPosition`**
 : null | Variable of type: themePlacement | Variable value of type: themePlacement

**`textOpacity`**
 : null | Variable of type: decimal | Variable value of type: decimal
