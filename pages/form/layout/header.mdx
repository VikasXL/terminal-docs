# formLayoutHeader

## update

```js filename="Terminal" copy
update formLayoutHeader showEnterprise:"true | false | yes | no"
update formLayoutHeader backgroundColor:"..."
update formLayoutHeader foregroundColor:"..."
update formLayoutHeader hyperlinkVars:"..."
update formLayoutHeader formNamePattern:"..."
update formLayoutHeader formNameFont:"..."
update formLayoutHeader formNamePosition:"..."
update formLayoutHeader formNameOpacity:"..."
update formLayoutHeader showSeparator:"true | false | yes | no"
```

**`backgroundColor`**
 : null | Variable of type: color | color value

**`foregroundColor`**
 : null | Variable of type: color | color value

**`hyperlinkVars`**
 : null | @empty | Variable1, Variable2, ...

**`formNamePattern`**
 : null | Variable of type: text | text value

**`formNameFont`**
 : null | Variable of type: font | font value

**`formNamePosition`**
 : null | Variable of type: themePlacement | themePlacement value

**`formNameOpacity`**
 : null | Variable of type: decimal | decimal value
