# formLayoutFooter

## update

```js filename="Terminal" copy
update formLayoutFooter showSeparator:"true | false | yes | no"
update formLayoutFooter backgroundColor:"..."
update formLayoutFooter foregroundColor:"..."
update formLayoutFooter line:"first | second | third" label:"first | second | third" textPattern:"..."
update formLayoutFooter line:"first | second | third" label:"first | second | third" textFont:"..."
update formLayoutFooter line:"first | second | third" label:"first | second | third" textOpacity:"..."
```

**`backgroundColor`**
 : null | Variable of type: color | Variable value of type: color

**`foregroundColor`**
 : null | Variable of type: color | Variable value of type: color

**`textPattern`**
 : null | Variable of type: text | Variable value of type: text

**`textFont`**
 : null | Variable of type: font | Variable value of type: font

**`textOpacity`**
 : null | Variable of type: decimal | Variable value of type: decimal
