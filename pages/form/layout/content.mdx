# formLayoutContent

## ensure

```js filename="Terminal" copy
ensure formLayoutContent FormLayoutContentName
```


## remove

```js filename="Terminal" copy
remove formLayoutContent FormLayoutContentName
```

**`label`**
 : null | Variable of type: text | text value

**`displayLabel`**
 : null | Variable of type: text | text value

**`allowToSwitchLayouts`**
 : null | @empty | FormLayoutContent1, FormLayoutContent2, ...

**`fields`**
 : null | @empty | SectionField, Grid/Field, ...

**`contentLayouts`**
 : null | @empty | FormLayoutContent1, FormLayoutContent2, ...

**`gridLayouts`**
 : null | @empty | Grid1/Layout1, Grid2/Layout2, ...
